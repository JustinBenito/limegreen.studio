# Coolvetica Font

Please download the Coolvetica font and place `coolvetica.woff2` in this directory.

You can find Coolvetica font from:

- https://www.dafont.com/coolvetica.font
- Or search for "Coolvetica font download"

After downloading:

1. Convert the font to .woff2 format if needed (use https://cloudconvert.com/ttf-to-woff2)
2. Place the file as `coolvetica.woff2` in this directory
3. The font is already configured in the project and will work once the file is added

The font is currently used for:

- Hero section headings
- Process section titles
- Start Shipping section heading
- Footer branding
